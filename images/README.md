# Portfolio Images

## Image Guidelines

Add your project images to this folder to display in your portfolio.

### Image Naming Convention:

**AJ Events:**
- `ajevents-1.jpg` - Event Setup / Production
- `ajevents-2.jpg` - Client Activation
- `ajevents-3.jpg` - Venue Design

**In Bloom:**
- `inbloom-1.jpg` - Community Gathering
- `inbloom-2.jpg` - Panel Discussion
- `inbloom-3.jpg` - Workshop Setting

**The Curated Edit:**
- `curated-1.jpg` - Concept Mood Board
- `curated-2.jpg` - Installation Design
- `curated-3.jpg` - Experience Flow

### Recommended Image Specs:

- **Format:** JPG or PNG
- **Dimensions:** 1200px x 800px (3:2 ratio) or larger
- **File Size:** Under 500KB for optimal loading
- **Quality:** High resolution, professionally shot or curated

### Tips:

- Use neutral, luxury-aesthetic images that match the portfolio vibe
- Ensure images are properly licensed or your own work
- Consider using tools like Unsplash or Pexels for placeholder images
- Images should reinforce the brand story and project impact

### To Add Images:

1. Place your images in this `/images` folder
2. Update the HTML in `index.html` by uncommenting the `<img>` tags
3. Example:
   ```html
   <!-- Replace this: -->
   <span class="project-image-placeholder">Event Setup</span>
   
   <!-- With this: -->
   <img src="images/ajevents-1.jpg" alt="AJ Events Setup">
   ```

The portfolio will display elegant placeholder text until you add your images.
